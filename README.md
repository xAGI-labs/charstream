# Charstream.xyz

Charstream.xyz is an AI-powered chat application that allows users to create and interact with custom AI characters. Whether you want to chat with your favorite fictional personalities, create entirely new characters, or explore AI-powered storytelling, Charstream.xyz provides a platform to bring your imagination to life.

## Features

- **Create AI Characters**: Design your own AI characters with customizable attributes like name, avatar, and personality.
- **Chat with Characters**: Engage in conversations with AI characters in text, voice, or video modes.
- **Ghiblify Your Images**: Transform your photos into Studio Ghibli-style avatars using the Ghibli image generator.
- **Interactive Storytelling**: Use AI to create immersive storytelling experiences with your characters.
- **Memory Layer**: Characters can remember past interactions, making conversations more personalized.

## Key Components

- **Chat Modes**: Switch between text, voice, and video chat modes for a versatile user experience.
- **Character Creation**: Use the character creation section to design and customize AI characters.
- **Voice and Video Chat**: Leverage advanced voice and video capabilities for more engaging interactions.
- **Ghibli Image Generator**: A dedicated section for creating Studio Ghibli-inspired avatars.
- **Discover Page**: Explore popular and educational characters created by the community.

## Technologies Used

- **Next.js**: A React framework for building the application.
- **Prisma**: For database management and querying.
- **Clerk**: For user authentication and management.
- **Tailwind CSS**: For styling and responsive design.
- **Lucide Icons**: For modern and customizable icons.
- **PostHog**: For real-time analytics and user tracking.
